# 变量配置

## 填充对象

```java
@ConfigurationProperties(prefix = "redis.props")
public class RedisProperties {
    Integer maxIdle;
    Integer minIdle;
}
```

```java
@PropertySource("classpath:redis-conf-${spring.profiles.active}.properties")
@ConfigurationProperties(prefix = "redis.props")
public class RedisProperties {}
```

```java
@Component
public class RedisProperties {
    @Value("${redis.props.maxIdle}")
    Integer title;
}
```
